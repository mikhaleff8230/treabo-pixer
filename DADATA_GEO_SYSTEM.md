# Гибридная система геолокации на базе DaData.ru

## Новая архитектура (приоритет)

1. **DaData.ru** - основной сервис для определения местоположения по IP и поиска городов
2. **MaxMind** - fallback для базовой геолокации
3. **Yandex** - опционально (код сохранен, но отключен по умолчанию)

## Что нужно настроить

### 1. Данные для DaData API

Нужны:
- `DADATA_API_KEY` - API ключ
- `DADATA_SECRET_KEY` - секретный ключ

### 2. Возможности DaData для геолокации

DaData предоставляет:
- **Определение города по IP** - через API `https://suggestions.dadata.ru/suggestions/api/4_1/rs/iplocate/address`
- **Поиск городов** - через API `https://suggestions.dadata.ru/suggestions/api/4_1/rs/suggest/address`
- **Валидация адресов** - уже реализовано
- **Поиск по КЛАДР** - уже реализовано

## План реализации

1. ✅ DaData клиент уже инициализирован
